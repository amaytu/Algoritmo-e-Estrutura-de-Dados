1- O que é Estrutura de Dados?
R: é uma maneira de organizar e listar os dados, de modo que permita o acesso de forma eficiente e a manipulação eficaz

2-Qual sua importâcia?
R: é importante para tornar qualquer programa mais Eficiente, simples e organizado para se manipular

3-Explique e exemplifique as estruturas Array (Estático e dinâmico), lista, fila e pilha:
R:

Array: Um array é uma estrutura de dados fundamental em programação que permite armazenar uma coleção de elementos do mesmo tipo sob um único nom
Estático: Ao criar um array, pré-definimos a quantidade máxima de registros.
Dinâmico: Existe um método para aumentar a capacidade dinamicamente

Lista: Esta interface define um contrato para uma lista ordenada de elementos, onde cada elemento tem um índice associado. 

Fila: O padrão fila ou queue é uma estrutura de dados que segue o princípio "primeiro a entrar, primeiro a sair" (FIFO - First-In-First-Out).

Isso significa que o primeiro elemento inserido na fila será o primeiro a ser removido, esse é um dos padrões mais utilizados nos projetos

Pilha: O padrão de pilha (Stack) em programação segue o princípio "último a entrar, primeiro a sair" (LIFO - Last-In-First-Out). Isso significa que o último elemento adicionado à pilha será o primeiro a ser removido.
